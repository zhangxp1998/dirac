# Speed and Performance

**Countless context curation optimizations allow Dirac to run lean and fast.**

To prevent "context bloat," Dirac uses surgical discovery tools like `get_file_skeleton` to map out your project's structure without reading every line. This "just-in-time" code loading keeps the context window focused, improving LLM performance and reducing latency.
